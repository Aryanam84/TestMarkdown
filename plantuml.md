### Umgebungsvariable, Environment Variable
 Damit PlantUml gerendert werden kann ist eine Java Umgebungsvariable nötig

>JAVA_HOME= C:\Program Files\Java\jdk-14.0.1  
>Unter Path ist dies zu ergänzen:  %JAVA_HOME%\bin
